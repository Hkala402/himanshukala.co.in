# himanshukala.co.in

Personal portfolio website for Himanshu Kala — AI QA Engineer.

Hosted on [GitHub Pages](https://pages.github.com/) at [www.himanshukala.co.in](https://www.himanshukala.co.in)

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main portfolio page (all sections) |
| `style.css` | All site styles |
| `script.js` | Typewriter, nav, animations, certificate viewer |
| `CNAME` | Custom domain mapping |
| `Himanshu_Kala_Resume.pdf` | Resume PDF (linked from site) |
| `_config.yml` | Jekyll config (for blog posts) |

## Sections

- About Me
- Skills & Expertise
- Work Experience
- Education (with DUCAT certificate viewer)
- Projects
- Achievements
- Contact

## Deployment

Push any file to the `main` branch — GitHub Pages deploys automatically.
