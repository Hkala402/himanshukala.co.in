Who I Am

I'm an AI Quality Assurance Engineer at Simublade Technology, working at the intersection of language model outputs and real-world reliability. My day-to-day involves manually testing AI application responses against context data, validating for factual accuracy, contextual relevance, and detecting hallucinations before they reach users.

I specialize in RAG pipeline validation — testing retrieval-augmented generation systems end-to-end, from context injection to output faithfulness. I also run model regression tests across version updates to catch subtle behavioral shifts that standard metrics miss.

Outside of work, I build portfolio tools that showcase AI input validation and QA thinking — because the best way to demonstrate testing skill is to build something testable.
